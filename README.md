# Dynamic Typing Effect

```html
<script>
let text = "Welcome to My Portfolio!"
let index = 0;
function type() {
    if (index < text.length) {
        document.getElementById("dynamicText").innerHTML += text.charAt(index);
        index++;
        setTimeout(type, 100);
    }
}
type();
</script>
```
